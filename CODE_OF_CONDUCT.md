# Contributor Code of Conduct

## Our pledge

We want MP5 Audio to be a welcoming place for contributors, reviewers, and users. Everyone participating in this project is expected to treat others with respect and professionalism.

## Expected behavior

- Be kind, constructive, and patient — especially with newcomers.
- Give actionable feedback in reviews and issues.
- Accept good-faith disagreement; focus on technical merit and project goals.
- Respect boundaries: no harassment, discrimination, personal attacks, or unwanted contact.
- Keep discussions on-topic and honest about the project's alpha / experimental status.

## Unacceptable behavior

- Harassment, hate speech, or discriminatory language.
- Trolling, insults, or sustained disruption.
- Publishing others' private information without consent.
- Sexualized language or imagery, or unwelcome sexual attention.
- Other conduct that would be inappropriate in a professional community.

## Enforcement

Project maintainers may remove, edit, or reject contributions and temporarily or permanently ban participants who violate this code of conduct.

Report concerns to the repository owner via GitHub (profile message) or, for security-related conduct tied to vulnerability reports, through the process in [`SECURITY.md`](SECURITY.md).

## Scope

This code applies in project spaces (GitHub issues, PRs, discussions) and when someone represents the project in public.
