### Default preset: **High v4**

Keep High as converter default unless ear tests show Extreme is clearly preferable for your workflow.

### Is Standard v4 acceptable?

**Yes** — user-confirmed major improvement over v3. Metrics: +8.1 dB SNR, 12× fewer at-limit sample pairs vs v3, only ~1% smaller file than High. Use Standard when file size matters slightly more than the last bit of cleanliness; use **High** for a primary listening library.

### Should High remain the default?

**Yes.** High adds +3.8 dB SNR over Standard for ~1.2% larger file, eliminates bench clipping on ORIGAMI, and matches the converter default. Extreme is +3.5 dB SNR for ~0.8% more size — choose Extreme only when that last margin is worth it.
