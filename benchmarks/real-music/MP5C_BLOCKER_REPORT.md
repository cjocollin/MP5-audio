## Scaling / conversion note

Encode uses i16/32768 → float; decode uses float*32767 → i16 (asymmetric full-scale). MP5-L bypasses this quant loop. Residual noise on non-silent content is dominated by scalar quant step, not pack modes.

## Transparency gate criteria

- [ ] Silence decodes as silence (bit-exact)
- [ ] Quiet passages: no obvious hiss vs MP5-L
- [ ] Extreme headphone-clean by ear
- [ ] Worst-window metrics pass

**Current status:** MP5-L and PCM pass; MP5-C fails listening bar.

## Recommended policy

- **Default export:** MP5-L (lossless)
- **Reference:** PCM fallback
- **MP5-C:** experimental only — may add hiss on all presets
- **MP5-H:** disabled until MP5-C base is clean (uses MP5-C + residual)
